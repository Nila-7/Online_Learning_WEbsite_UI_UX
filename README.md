Online Learning Web Application UI/UX Design

## 📌 Project Overview
This project is a modern Online Learning Web Application UI/UX design created using Figma. The design focuses on creating a clean, responsive, and user-friendly educational platform for students and learners.

The project demonstrates the application of UI/UX design principles such as layout structure, typography, color theory, responsiveness, visual hierarchy, and interactive prototyping.

---

## 🎯 Objectives
- Create a modern educational website interface
- Improve user experience and accessibility
- Design responsive layouts for desktop screens
- Provide easy navigation between pages
- Build an attractive and organized learning platform

---

## ✨ Features
- Responsive Landing Page
- Student Dashboard
- Course Listing Section
- Course Details Page
- User Profile Section
- Interactive Navigation
- Modern UI Components
- Clean Typography & Color Palette
